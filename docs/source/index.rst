***********************
StarbugII Documentation
***********************

PSF photometry suite for complex and crowded regions in astronomical images.


.. important::
    - If you use *StarbugII* in any of published or presented work, please include a :doc:`citation <./ref>`.
    - *StarbugII* makes use of `astropy <https://docs.astropy.org/en/stable/>`_ and `photutils <https://photutils.readthedocs.io/en/stable>`_, please acknowledge them accordingly.
